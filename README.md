### PM2 WebUI
Opensource Alternative to PM2 Plus

##### USAGE
```
git clone https://github.com/suryamodulus/pm2-webui
cd pm2-webui
npm install
cp env.example .env
npm start
```
> Note : Please change SESSION_SECRET and APP_PASSWORD in .env file to something more secure
